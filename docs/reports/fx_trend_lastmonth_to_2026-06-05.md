比較軸: ① 月次 (March→April→May) ② 週次 (5/11週→今週) ③ 今週 vs 直近1ヶ月の差分
作成日: 2026-06-06

> 用語: 月次レビューは各月1日生成で**直前約20営業日**を対象 → `202606` = 5月分が「直近1ヶ月」。
> 週次は report 日付の前週 (`20260606` = 6/1〜6/5 = 今週)。

## サマリー (TL;DR)

- **誤差 (MAE) が一貫して縮小し、今週が観測史上ベスト**。UGH 週次 MAE は 5/11週 ~36bp → 今週 **~7.0bp** (約 **−80%**)。
- **直近1ヶ月 (5月) → 今週で全指標が改善**: 方向 66.7%→**75%**、終値 MAE ~20bp→**~7bp (約−65%)**、state hit 73–87%→**100%**。
- 月次の質的転換: 3月 (v1) は MAE 40bp + 2フラグ → 4月 (v1) は方向 31% に劣化し「baseline 優位」フラグ → **5月 (v2, 4バリアント) で初の "keep_current_logic"（フラグゼロ）**。
- この改善は **engine review 2026-05 のチューニング (v2.1 state sharpening → v2.3 per-variant range / magnitude 較正)** が 5月末〜6月頭に landing した時期と一致。
- 注意点: range_hit は週次で 100%→**75%** に「低下」して見えるが、これは ensemble 一括 100% から **per-variant の正直な値 (P3B/v2.3)** への計測方式是正であり、実質劣化ではない。

## 4. トレンド総括

1. **改善は本物かつ engine 変更と因果整合** — MAE の構造的低下 (40→7bp) と「フラグ有り→keep_current_logic」への遷移が、v1→v2 化および v2.1–v2.3 チューニング (state sharpening / per-variant range / magnitude 較正) の landing 時系列と一致。
2. **方向精度は依然ボラタイル** — 週次 37.5–75% で振れる。今週は上限だが、レジーム依存 (trending 100% / choppy 0%、5月月次より) が残る。トレンド相場特化の傾向は不変。
3. **state lifecycle が稼働開始** — 5月に dormant/fire/setup へ多様化、fire state が高品質シグナル化。これは売買レイヤー planning の有力入力。
4. **計測の正直化** — range_hit が ensemble 100% から per-variant 75% へ。見かけの低下だが過大評価の是正であり、信頼できる指標になった。

### 監視ポイント (次週以降)
- 方向のボラティリティ低減 — choppy 相場での 0% を埋められるか。
- fire state の出現頻度と的中の持続性。
- range_hit が per-variant 75% 前後で安定するか (v2.3 較正目標 ~72% 近傍)。

> データ品質: 比較対象全期間で provider 障害ゼロ (alpha_vantage)、fallback 発火なし。月次は missing window が 3月65%→5月25–33% へ減少し、データ被覆も改善傾向。
